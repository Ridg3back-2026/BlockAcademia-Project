How It Works

1

Credential Creation

Educational institutions or authorized issuers create a credential with student details and achievements. The data is hashed using SHA-256 encryption.

2

Blockchain Storage

The credential hash and issuer's public key are recorded in a new block and added to the blockchain, creating a permanent, tamper-proof record.

3

Verification Process

Anyone can verify a credential by providing the hash and issuer's public key. The system checks the blockchain to confirm authenticity.

4

Trust & Transparency

The decentralized nature ensures no single entity can manipulate records, providing complete transparency and trust in the verification process.
